How to set up your static site

  • Place your site files in public/local/sites/main/
  • Include an index.html at the root of that directory
  • Static assets (CSS, JS, images) go alongside it in the same folder tree
  • Restart the service after changes: fzdocker public run --force --persist

The site is served at the root URL. Additional sites can be added under public/local/sites/ and configured in public/nginx.conf.tpl.
